---

## 📝 How I Use This Dev Log

Each log entry contains:


## Today’s Focus
- Quick bullet of main tasks

## Wins / What I Learned
- Short notes on progress or insights

## Challenges / Bugs
- Quick description of any problems
- Next steps to fix

## Tomorrow / Next Steps
- 1-3 bullets of what to tackle next
